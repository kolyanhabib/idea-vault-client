## 📌 About The Project

**IdeaVault** is a modern full-stack idea sharing platform where users can explore, create, and manage innovative ideas across different categories.

The platform provides a smooth and responsive experience with authentication, idea management, category filtering, and a clean modern UI. Users can securely share their creativity and discover ideas from others.

## ⚙️ Installation

```bash
git clone https://github.com/kolyanhabib/idea-vault
cd idea-vault
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file and add:

```env
MONGODB_URI=your_mongodb_uri

BETTER_AUTH_SECRET=your_secret
BETTER_AUTH_URL=http://localhost:3000

GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```
